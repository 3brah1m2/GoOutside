# GoOutside

GoOutside is an automated monitoring project that determines whether it is **safe**, **use caution**, or **unsafe** to go outside in **Karachi** based on real-time air quality and weather data.

It sends notifications to a Discord server using webhooks and runs automatically via GitHub Actions.
